Q. What happens when you initialize a repository? Why do you need to do it?

A: When I run git init command, a new empty git repository will be initialized? I need to do it because I want to create a new repository in a desired location. In addition I can also run git status to see the initial commit.

Q. How is the staging area different from the working directory and the repository? What value do you think it offers?

A: Staging area is different from the working directory and the repository in the sense that it stages or unstages the file.
